# PROJETO - IMOBILIÁRIA
Front-End de imobiliária com foco no aprendizado em utilizando react.js, redux toolkit, axios, tailwind css, docker e react router.


````bash
src/
├── assets/           # Imagens/fontes
├── components/       # Componentes reutilizáveis
│   ├── ui/           # Botões, cards, inputs
│   └── layout/       # Header, Sidebar
├── features/         # Módulos do CRM
│   ├── dashboard/
│   ├── leads/
│   ├── properties/
│   ├── calendar/
│   ├── tasks/
│   └── reports/
├── services/         # Lógica de negócio/API
├── hooks/            # Hooks customizados
├── store/            # Configuração Redux
├── utils/            # Funções utilitárias
├── App.jsx           # Componente raiz
└── index.js          # Ponto de entrada
```
