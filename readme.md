# 🏡 Real Estate CRM

Um sistema de gerenciamento de relacionamento com clientes focado no mercado imobiliário, desenvolvido com React 18 e containerizado com Docker.

## 🚀 Tecnologias

- **React 18** - Framework principal
- **Docker** - Containerização e ambiente de desenvolvimento
- **ESLint** - Padronização e qualidade de código
- **React Router** *(em implementação)* - Roteamento da aplicação
- **TailwindCSS** *(em implementação)* - Framework CSS utilitário

## 📦 Como executar

### Usando NPM
```bash
npm install
npm start
```

### Usando Docker
```bash
docker-compose up --build
```

## 📁 Estrutura do projeto

```
src/
├── features/
│   ├── dashboard/     # Dashboard principal
│   ├── leads/         # Gestão de leads
│   └── properties/    # Gerenciamento de imóveis
├── components/        # Componentes reutilizáveis
├── services/          # Serviços de API
└── store/            # Gerenciamento de estado
public/
└── index.html        # Template base
```
